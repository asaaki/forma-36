### Content and interaction Guidelines

- Depending on the shown information the right width configuration should be applied. Pure form views should be limited to 768px for enhanced readability. Simple table views should default to a maximum width of 1240px. Rich data views like dashboards or configurable tables should use the entire available width.

### Dimensions and Specs

| Property            | Value               |
| :------------------ | :------------------ |
| Width: Default      | 1240px, 786px, 100% |
| Container Alignment | center              |
| Content Alignment   | left                |
| Padding             | spacing-l           |
